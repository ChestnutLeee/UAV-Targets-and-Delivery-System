# UAV-Targets-and-Delivery-System
Design of a Machine Vision-Based UAV Dynamic Target Precision Recognition and Delivery System
    In response to the issue of precise identification and delivery of dynamic targets by drones in GPS-denied environments, this study has designed and implemented a drone dynamic target precise identification and delivery system based on machine vision. Innovative points such as the nested dual-scale composite tag (NDCT) design and a real-time visual perception pipeline integrated with EVSR super-resolution are proposed. Simulation results show that all performance indicators of the system meet or exceed the design requirements, and actual machine tests have verified the system's engineering practicality.
